What "paste" does for a file item:
  • Writes the file to the clipboard as a public.file-url (native macOS file reference).
  • Press ⌘V in Finder → file copied. In Mail/Slack/Messages → attached. In a text field → path inserted.
  • Press ⌘1–⌘9 on the grid → same paste action, no opening the popover.
  • Click the share icon → real NSSharingServicePicker with the file (AirDrop, Mail, Notes, etc.) — no extra work, the existing helper just receives a file path instead of text.
Storage: Files are copied into ~/Library/Application Support/restash/files/ on save. The item stores the path + original name + size + MIME type. Moving or deleting the source file never breaks the item.
