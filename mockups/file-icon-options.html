Restash — File icon set

Three SVG icon directions for file items, each covering the same nine file kinds (generic, image, video, audio, PDF, document, spreadsheet, presentation, archive). Pick one and I'll wire it into the live app.

Style 1 Document with type label

A document silhouette with the file extension lettered inside. Unmistakable at small sizes; each kind reads as a real "document with a thing on it" — what Finder and the Files app trained everyone to expect.

Same document outline, but instead of text labels each shows a mini glyph of what's inside (mountain for image, play triangle for video, music notes for audio). More visual / less typographic.
