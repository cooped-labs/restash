Restash — QR decoder

See a QR code on screen? Hit a hotkey, drag a box around it, and Restash decodes it in place — no phone, no camera, no extra app. The preview tells you exactly what's encoded and lets you act on it.

Implementation notes

Hotkey + capture

  • Default: ⌘⇧Q (configurable in Settings, same recorder as the main hotkey).
  • For region selection, shell out to screencapture -i -s (built-in macOS) — captures the region as a temp PNG and returns. Free, zero deps, same UX as ⌘⇧4.
  • Cancellation is automatic (screencapture returns nothing on Esc).

Decode

  • Native: VNDetectBarcodesRequest from macOS Vision framework via a small Swift helper. Detects QR + many other formats (Code 128, EAN, Data Matrix, …).
  • JS fallback: jsqr npm package in the main process if Vision is unavailable.
  • If no QR found in the captured region → friendly "No QR code found" preview with a "Try again" button.

Content detection

  • URL: starts with http(s):// or has a recognized TLD. Strip tracking params on copy/save (existing toggle).
  • Wallet: regex match per chain. SOL = 32–44 base58. ETH = 0x + 40 hex. BTC = bc1…, 1…, or 3….
  • Wi-Fi: starts with WIFI:. Parse S:, T:, P: fields.
  • vCard: starts with BEGIN:VCARD. Parse name + phone + email.
  • Otherwise: plain text.

Save flow

  • "Save" appends to the user's currently-selected stash (same default as the main editor).
  • Pre-fills label based on detected type — URL → site title (fetched on save); Wallet → chain name; Wi-Fi → SSID; Text → first 40 chars.
