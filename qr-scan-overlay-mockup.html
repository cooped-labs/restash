Restash — QR scan overlay

Three directions for what shows up after you hit ⌘⇧F. Today it's just the stock macOS region selector. Each option below replaces that with a Restash-branded scanning UI so the action feels like "scanning a QR", not "taking a screenshot."

Option A — recommended Spotlight Lens

Dim overlay everywhere except a circular "lens" that follows your cursor. Hover a QR → animated scan-line crosses the lens → auto-snaps the moment Vision finds a QR. No clicking, no dragging. Most magical.

Implementation notes

All three replace screencapture -i -s with a fullscreen transparent Electron window that draws the overlay itself.
  • Option A / B need a continuous capture loop — every ~250 ms, grab a low-res screenshot of the display(s) and run VNDetectBarcodesRequest on it. The overlay window stays on top but transparent so the user sees the real screen underneath. macOS Screen Recording permission required.
  • Option C is the lightest — no continuous scan, no Screen Recording permission needed up front (only at capture time). Just a styled selection box.
  • Cancellation: esc closes the overlay window in all three.
Recommendation: ship C first (one-day change) for the brand win, then add A as a Pro-tier upgrade later — the auto-detect magic is a great paid-feature hook.
